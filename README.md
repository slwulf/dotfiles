### setup steps
1. clone this repo into the home dir
 - `git clone git@github.com:slwulf/dotfiles.git .dotfiles`
1. make sure the shell is bash
1. run `~/.dotfiles/scripts/setup.sh`
1. on mac, install (monokai terminal)[https://github.com/stephenway/monokai.terminal]
1. for vscode, base prefs and keyboard shortcuts are in a private gist (TODO move these into the repo)
